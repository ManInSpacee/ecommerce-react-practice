# Ecommerce React Project

This project is a comprehensive e-commerce application built using React as part of the SuperSimpleDev course. It demonstrates the key concepts of building scalable frontend applications.

## Project Structure

```
/ecommerce-react-practice
├── /public
│   ├── index.html
│   └── favicon.ico
├── /src
│   ├── /components
│   │   ├── Header.js
│   │   ├── Footer.js
│   │   └── ProductList.js
│   ├── /pages
│   │   ├── Home.js
│   │   ├── Product.js
│   │   └── Cart.js
│   ├── App.js
│   └── index.js
└── package.json
```

## Setup Instructions

1. **Clone the Repository**  
   Clone this repository to your local machine:
   ```bash
   git clone https://github.com/ManInSpacee/ecommerce-react-practice.git
   cd ecommerce-react-practice
   ```

2. **Install Dependencies**  
   Install the necessary packages:
   ```bash
   npm install
   ```

3. **Run the Application**  
   Start the application:
   ```bash
   npm start
   ```
   The app will be running at `http://localhost:3000`.

## Features

- **Product Listing**: Browse through a list of products with filtering options.
- **Product Details**: View detailed information about each product.
- **Shopping Cart**: Add products to a cart and view your selections.
- **Responsive Design**: Optimized for both desktop and mobile views.
- **User Authentication**: Secure sign-in and sign-up forms (to be implemented).
