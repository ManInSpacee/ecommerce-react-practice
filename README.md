# Проект электронной коммерции на React

Этот проект является учебным примером платформы электронной коммерции, разработанным с использованием библиотеки React. Цель данного проекта — продемонстрировать основные концепции React и дать представление о построении современного веб-приложения.

## Функциональные возможности

- **Каталог продуктов:** Просмотр широкого спектра товаров с возможностью фильтрации и сортировки.
- **Корзина:** Легкий процесс добавления и удаления продуктов из корзины.
- **Оформление заказа:** Удобный интерфейс для ввода платежной информации и завершения покупки.
- **Аутентификация:** Пользователи могут создавать учетные записи и входить в систему для управления своими заказами.

## Технологии, использованные в проекте

- React
- Redux для управления состоянием
- React Router для навигации
- Axios для выполнения HTTP-запросов

Этот проект является отличной основой для изучения React и разработки более сложных приложений электронной коммерции.
